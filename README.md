# BA_logo

From https://github.com/nulla2011/bluearchive-logo

本项目为原项目的改编版本。著作权归原作者 nulla2011 所有，若有侵权请联系删除。

## 运行方法

运行 `run.py`，之后打开 `http://localhost:8000/`。你也可以根据需要在该 Python 文件中自行调整映射。

部署在 Web 网站中则不需要运行 `run.py`。

## 在线地址

[GitHub Pages Demo](https://gudong2003.github.io/BA_logo/)

## 当前版本功能

- 主标题左右文本输入（`Blue` / `Archive`）
- 副标题输入与开关（默认关闭）
- 透明背景开关
- 高级设置默认展开
- 光环位置微调（`X / Y`）
- 分辨率缩放（用于 PNG 导出清晰度）
- PNG 导出（文件名包含分辨率信息）
- SVG 导出（矢量导出，独立于分辨率缩放）
- 一键复制 PNG 到剪贴板
- 中英文界面文案

## 导出说明

- PNG：受分辨率缩放影响，缩放越大导出像素越高。
- SVG：矢量导出，不受分辨率缩放影响。
